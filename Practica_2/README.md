# Веб-сервер на Go

## Запуск проекта с GitHub

### 1. Скачайте проект с GitHub
Откройте командную строку и выполните:
```bash
git clone https://github.com/MrFandore/Goland.git
cd Goland/Practica_2
```

### 2. Проверьте файлы проекта
Убедитесь, что в папке `Practica_2` есть следующие файлы:
- `main.go` - главный файл программы
- `app.go` - логика сервера  
- `logger.go` - система логирования
- `go.mod` - конфигурация проекта

### 3. Запустите сервер
```bash
go run main.go
```

**Если видите эту ошибку:** `'go' is not recognized as an internal or external command`  
**Значит Go не установлен на компьютере!**

## Что делать если Go не установлен

### Вариант 1: Установить Go (обязательно!)
1. Скачайте Go с официального сайта: https://golang.org/dl/
2. Установите как обычную программу
3. Перезапустите командную строку
4. Проверьте установку: `go version`

### Вариант 2: Попросить кого-то с установленным Go
Проект можно запустить только на компьютере с установленным Go.

## После установки Go:

### Запуск сервера:
```bash
go run main.go
```

**Успешный запуск:** Вы увидите сообщение:
```
[INFO] 2024-01-01T12:00:00Z Server is starting on :8080
```

### Проверка в браузере:
1. Откройте браузер
2. Перейдите по адресу: `http://localhost:8080/`
3. Должен появиться текст: `Hello, Go project structure!`

### Проверка статуса сервера:
1. Перейдите по адресу: `http://localhost:8080/ping`  
2. Должен появиться JSON с текущим временем

## Если возникают проблемы:

**Порт занят:** Измените порт в файле `main.go` с `:8080` на `:8081`

**Файлы не найдены:** Убедитесь, что вы в папке `Goland/Practica_2`

**Go не работает:** Переустановите Go и перезагрузите компьютер

## Остановка сервера:
Нажмите `Ctrl+C` в окне командной строки

---

**Важно:** Без установленного Go запустить проект невозможно! Это требование языка программирования.
