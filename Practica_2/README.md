# Веб-сервер на Go

## Запуск проекта

### 1. Откройте командную строку
- На Windows: нажмите Win+R, введите `cmd`, нажмите Enter
- На Mac: откройте Terminal через поиск
- На Linux: откройте Terminal

### 2. Перейдите в папку с проектом
```bash
cd путь_к_вашей_папке_с_проектом
```
Например:
```bash
cd C:\Users\ИмяПользователя\practica2
```

### 3. Запустите сервер
```bash
go run main.go
```

**Результат:** Вы должны увидеть сообщение:
```
[INFO] 2024-01-01T12:00:00Z Server is starting on :8080
```

## Проверка работы

### Способ 1: Проверка в браузере

1. **Откройте браузер** (Chrome, Firefox, Edge)
2. **Введите адрес:** `http://localhost:8080/`
3. **Должен появиться текст:** `Hello, Go project structure!`

4. **Проверьте второй адрес:** `http://localhost:8080/ping`
5. **Должен появиться JSON:** 
```json
{"status":"ok","time":"2024-01-01T12:00:00Z"}
```

### Способ 2: Проверка через командную строку

1. **Откройте новое окно командной строки** (не закрывая сервер)
2. **Выполните команды:**

```bash
curl http://localhost:8080/
```
**Результат:** `Hello, Go project structure!`

```bash
curl http://localhost:8080/ping
```
**Результат:** JSON с статусом и временем

## Проверка логов

**В окне где запущен сервер** вы должны видеть записи при каждом запросе:
```
[2024-01-01T12:00:00Z] [127.0.0.1] GET /
[2024-01-01T12:00:01Z] [127.0.0.1] GET /ping
```

## Что должно работать:

✅ **Главная страница** (`http://localhost:8080/`) - текст "Hello, Go project structure!"  
✅ **Статус сервера** (`http://localhost:8080/ping`) - JSON с временем  
✅ **Логирование** - в консоли видны все запросы  
✅ **Сервер отвечает** - статус 200 на все запросы  

## Если не работает:

1. **Порт занят** - измените порт в файле `main.go` с `:8080` на `:8081`
2. **Файлы не найдены** - убедитесь, что вы в правильной папке с файлом `main.go`
3. **Go не установлен** - без установленного Go проект не запустить

## Остановка сервера
Нажмите **Ctrl+C** в окне командной строки где запущен сервер.

---

**Проект работает если:**
- В браузере открываются оба адреса
- В консоли сервера видны логи запросов
- JSON на /ping показывает текущее время
