# Практика 1 - HTTP Сервер на Go

## Структура проекта
```
Practica_1/
├── main.go
├── go.mod
├── go.sum
├── .gitignore
├── workspace.xml
├── Practica_1.iml
├── modules.xml
└── vcs.xml
```

## Файлы проекта
- **main.go** - основной код сервера
- **go.mod** - файл зависимостей Go
- **go.sum** - контрольные суммы зависимостей
- **.gitignore** - список игнорируемых файлов для Git

## Запуск проекта

### 1. Скачайте проект
```bash
git clone https://github.com/MrFandore/Goland.git
cd Goland/Practica_1
```

### 2. Установите зависимости
```bash
go mod download
```

### 3. Запустите сервер
```bash
go run main.go
```

## Проверка работы

После запуска сервер доступен по адресу: http://localhost:8080

### Проверка эндпоинтов:

**Главная страница:**
```
http://localhost:8080/hello
```
Ожидаемый результат: текст "Hello, world!"

**Информация о пользователе:**
```
http://localhost:8080/user
```
Ожидаемый результат: JSON с ID и именем пользователя

**Статус сервера:**
```
http://localhost:8080/health
```
Ожидаемый результат: JSON со статусом "ok" и временем

### Проверка через командную строку:
```bash
curl http://localhost:8080/hello
curl http://localhost:8080/user
curl http://localhost:8080/health
```

## Требования
- Установленный Go версии 1.25 или выше
- Установленный Git

## Решение проблем

**Порт занят:**
Измените порт в файле main.go с ":8080" на ":8081"

**Go не установлен:**
Скачайте и установите Go с https://golang.org/dl/

**Файлы не найдены:**
Убедитесь, что вы в папке Practica_1 с файлом main.go

## Остановка сервера
Нажмите Ctrl+C в командной строке где запущен сервер
