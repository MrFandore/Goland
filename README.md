# HTTP Server на Go

## Описание проекта
Это простой веб-сервер на языке программирования Go, который отвечает на запросы по трем адресам.

## Что нужно для запуска

### Установите Go
1. Скачайте Go с официального сайта: https://golang.org/dl/
2. Установите как обычную программу
3. Откройте PowerShell (Windows) или Терминал (Mac/Linux)
4. Проверьте установку командой: `go version`
5. Если видите версию Go - установка прошла успешно

### Установите Git
1. Скачайте Git с официального сайта: https://git-scm.com/
2. Установите как обычную программу
3. Проверьте установку командой: `git --version`

## Как запустить проект

### Шаг 1: Скачайте проект
```bash
git clone https://github.com/ваш-username/ваш-репозиторий.git
cd ваш-репозиторий
```

### Шаг 2: Установите зависимости
```bash
go mod download
```

### Шаг 3: Запустите сервер
```bash
go run main.go
```

Вы увидите сообщение: `Starting on :8080 ...`

## Как проверить работу

Сервер запущен и работает на адресе: http://localhost:8080

### Проверка 1: Приветствие
Откройте браузер и перейдите по адресу:
```
http://localhost:8080/hello
```
Должен появиться текст: `Hello, world!`

### Проверка 2: Информация о пользователе
Перейдите по адресу:
```
http://localhost:8080/user
```
Должен появиться текст в формате JSON с уникальным ID.

### Проверка 3: Статус сервера
Перейдите по адресу:
```
http://localhost:8080/health
```
Должен появиться статус "ok" и текущее время.

## Альтернативный способ запуска

### Создание исполняемого файла
```bash
go build -o server.exe main.go
./server.exe
```

### Изменение порта (если порт 8080 занят)
```bash
# Для Windows
$env:APP_PORT="9090"
go run main.go

# Для Mac/Linux
export APP_PORT=9090
go run main.go
```
После этого используйте адрес: http://localhost:9090

## Файлы в проекте
- `main.go` - основной код программы
- `go.mod` - файл с настройками проекта
- `go.sum` - файл с проверками безопасности
- `README.md` - эта инструкция

## Если что-то не работает
1. Убедитесь, что сервер запущен (видите сообщение "Starting on :8080")
2. Проверьте, что вводите правильные адреса
3. Если порт занят, измените порт как показано выше
4. Убедитесь, что Go установлен корректно (команда `go version` работает)
