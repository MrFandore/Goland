# Репозиторий Пряшникова Дмитрия Максимовича

## Описание репозитория

Данный репозиторий содержит практические работы по дисциплине **"Технологии создания программного обеспечения"** за 1 семестр 2025-2026 учебного года. Все проекты реализованы на языке программирования Go и демонстрируют различные аспекты разработки backend-приложений.

## Структура проектов

### Практика 1 - Базовый HTTP сервер
**Цель:** Изучение основ создания HTTP-сервера на Go
- Установка и настройка окружения Go
- Создание минимального HTTP-сервиса
- Работа с внешними зависимостями
- Базовые маршруты и JSON-ответы

### Практика 2 - Структура Go проекта  
**Цель:** Освоение правильной организации Go проекта
- Разделение кода по пакетам (cmd/, internal/, pkg/)
- Создание модульной архитектуры
- Логирование запросов
- Правильное импортирование пакетов

### Практика 3 - CRUD API с памятью
**Цель:** Реализация полноценного CRUD API
- Создание, чтение, обновление и удаление задач
- In-memory хранилище данных
- Middleware для логирования
- Валидация входных данных

### Практика 4 - Продвинутая маршрутизация с Chi
**Цель:** Изучение сторонних роутеров и middleware
- Использование Chi роутера
- Группировка маршрутов
- CORS middleware
- Полнофункциональное REST API

### Практика 5 - Работа с PostgreSQL
**Цель:** Изучение взаимодействия с реляционными базами данных
- Подключение к PostgreSQL с использованием database/sql
- Выполнение параметризованных запросов (INSERT, SELECT)
- Использование транзакций для массовых операций
- Настройка пула соединений и работа с контекстом
- Реализация репозитория для работы с задачами

## Технические особенности

Все проекты демонстрируют:
- **Чистую архитектуру** - разделение ответственности между компонентами
- **Правильную структуру** - организация кода по стандартам Go сообщества
- **Обработку ошибок** - корректные HTTP статусы и JSON-ответы
- **Логирование** - отслеживание запросов и ошибок
- **Валидацию** - проверку входных данных
- **Работу с базами данных** - использование PostgreSQL, настройка пула соединений, транзакции

## Требования для запуска

Для работы с проектами необходимо:
1. Установленный Go (версии 1.21 или выше)
2. Установленный Git
3. Любой текстовый редактор или IDE
4. Для Практики 5: установленный PostgreSQL

## Как использовать

Каждая практическая работа находится в отдельной папке (Practica_1, Practica_2 и т.д.) и содержит:
- Исходный код проекта
- Инструкции по запуску
- Примеры тестирования API
- Документацию по использованию

Для Практики 5 дополнительно требуется:
- Создать базу данных PostgreSQL
- Настроить соединение через переменные окружения
- Выполнить SQL-скрипт для создания таблицы задач

Для запуска конкретного проекта перейдите в соответствующую папку и следуйте инструкциям в README файле.
